# KinematicAdaptiveDBSAnalysis

Analysis code for Karjagi, Kehnemouyi et al. "Real-Time Kinematic Adaptive Deep Brain Stimulation Safely Reduces Gait Impairment and Freezing of Gait in Parkinson's Disease" (*in review*, 2026).

## Dependencies

**Python** (3.8+)
- numpy, pandas, matplotlib, seaborn

**R** (3.6+)
- lme4, lmerTest, emmeans, effectsize, dplyr, tidyr

*MATLAB**
- [Analysis-rcs-data](https://github.com/bronte-stewart-lab/Analysis-rcs-data) (for parsing Summit RC+S data)

## Data

Raw data are available from the corresponding author upon reasonable request.
